Ultimate Free Stuff Compendium

What is this?

The Ultimate Free Stuff Compendium (UFSC) is a compilation of products (primarily software) that are available for free. It will be updated as new products are discovered by me or submitted at the source code repo. Open source/source available products are preferred.

Key

- Open source. The source code is readily available or distributed with the product, and modification and redistribution is allowed.
- Source available. The source code is readily available or distributed with the product, but does not allow modification or redistribution.
- Closed source. The source code is not available.
- Contains features locked behind in-app payments, or subscriptions which add extra features or extend existing ones. Does not include software which is free for personal use but requires a license for commercial use.
