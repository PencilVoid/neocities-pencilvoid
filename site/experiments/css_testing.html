Information

This is an information box. It contains additional information that is not a warning and is not urgent. Some application examples include a notice about the content of a blog post or some important information about a form.

Warning

This is a warning box. It contains warning information and should be paid attention to. It should be used when it is important that the user knows the information it conveys. For example, it may be used if a form requires certain input or a widget only works in certain browsers.

Denied

This is an denial box. It indicates that some action has been denied. It should be used when an evaluation has been performed on the user agent and/or user input and it has determined that the attempted action cannot be carried out.

URGENT

This is an urgent box. It is like the warning box but more urgent. It should be used when the user must know the information it conveys as quickly as possible, and there is a high risk of substantial consequences if they do not.
