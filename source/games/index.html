Here is a small collection of games I made using Construct 2 back when I was newer to programming and game development. These games are dated from about 2018 according to their metadata, but they could be from earlier.

Note: these games require keyboard input and are designed for desktop or laptop devices. Additionally, Super Turret Shooter requires mouse input for gameplay and The Open Road requires mouse input for navigating menus.

Antivirus
A side-scrolling platformer/shooter. The levels quickly shift from actual gameplay to messing around.

Super Turret Shooter
A simple top-down shooter. There were experiments with switching between top-down and side-scrolling mid-level.

The Open Road
Top-down racing game in which the player competes with the clock.
